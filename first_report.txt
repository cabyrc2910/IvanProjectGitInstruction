Git init - Инициалтизация локального репозитория
Git status - получать ирформацию от программы о ёё текущем состоянии
Git add - добавить файл к следующему коммиту
Git comand -m - создание коммита
Git log - вывод на экран истории всех коммитов
Git checkout - переход от одного коммита к другому
Git checkout master - вернуться к актуальному состоянию и продолжить работу
Git diff - увидеть разницу между текущим файлом и закомиченным
